# Tutorials
Python tutorials created with Jupyter Notebook.

Goal of this tutorial is to provide some basic knowledge of Python to handle
genomics data.

You can open any of these tutorial through Google Colaboratory (Google Colab);

Google Colaboratory provides a great way to experiment with Python on the fly using your browser.
You do not need to download and run locally any jupyter notebook, you just need a Google Account.

You can install the Chrome Extension called [open in colab][1] to use these tutorials.

[1]: https://chrome.google.com/webstore/detail/open-in-colab/iogfkhleblhcpcekbiedikdehleodpjo?hl=en
